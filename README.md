# JJK Gacha Bot

A Discord bot built in Python for a JJK fanfic community server.

## Planned Features
- Gacha card system with canon and original characters
- Cursed Energy points earned by reacting to chapter announcements
- Battle system with unique mechanics per character
- Automated chapter announcements
- Custom role management

## Tech Stack
- Python
- discord.py
- Pytest (testing)
